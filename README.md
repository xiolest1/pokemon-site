## 🛠️ Installation

1. Clone the repository:
```bash
git clone https://github.com/xiolest1/pokemon-site.git
```

2. Navigate to the project directory:
```bash
cd pokemon-site
```

3. Open `index.html` in your browser or use a local server:
```bash
# Using Python
python -m http.server 8000
```
